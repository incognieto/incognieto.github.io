Scrapping

Pada era digital saat ini, informasi menjadi semakin mudah diakses melalui internet. Salah satu sumber informasi yang banyak digunakan adalah situs web berita. Namun, terkadang kita memerlukan akses cepat dan terstruktur terhadap data yang terdapat dalam situs web tersebut. Web scraping adalah teknik yang digunakan untuk mengekstrak data dari halaman web secara otomatis, yang dapat membantu dalam memperoleh informasi secara efisien dan teratur.

Web Scrapping : BeautifulSoup & Task Scheduler
Spesifikasi Program

  • Python: Bahasa pemrograman utama dengan dukungan library untuk web scraping dan pengolahan data.
  • BeautifulSoup: Library Python untuk mengekstrak data dari HTML dan XML.
  • JSON: Format penyimpanan data hasil web scraping.
  • Visual Studio Code: IDE untuk mengembangkan program Python.
  • Git: Sistem kontrol versi untuk mengelola perubahan dalam kode program.
  • GitHub: Platform hosting untuk menyimpan dan berbagi kode program.
  • Bootstrap: Framework untuk merancang tampilan web yang responsif.
  • XAMPP: Paket perangkat lunak untuk membuat lingkungan server web lokal.
  • Task Scheduler: Fitur pada sistem operasi Windows untuk menjadwalkan dan mengotomatisasi eksekusi tugas-tugas tertentu.

# Judul Berita Waktu Publish Kategori Tautan Waktu Scrapping

Tujuan dari praktikum untuk memenuhi penugasan dari mata kuliah Proyek 1 : Pengembangan Perangkat Lunak Program Studi D4 Teknik Informatika Politeknik Negeri Bandung, praktikum ini dilakukan untuk mempelajari dan mengimplementasikan teknik web scraping menggunakan BeautifulSoup pada situs berita Republika.co.id. Praktikum ini akan membahas langkah-langkah untuk melakukan scraping data dari situs tersebut dan menyimpannya dalam format yang mudah diakses dan diolah.
